用人单位缺席情况下，代理劳动者（高管）成功确认2021.12-2024.3期间劳动关系，裁决支付工资差额38万余元。

### 💰 萧某诉陈某追偿权纠纷案
代理原告成功追回代偿借款本金、利息、律师费及保全担保费合计约43万元，法院全额支持合理维权开支。

### 🔒 张某诉叶某隐私权纠纷案
代理原告获法院判决令被告停止传播、删除隐私视频、录制道歉录音，赔偿精神抚慰金1元及维权合理费用1.1万余元。

### 🚚 深圳某货运公司诉广州某货运公司多式联运合同纠纷
代理被告成功抗辩货物已交付且原告无实际损失，法院驳回原告32万余元全部索赔请求。
